# psihoterapija-kotnik

Vue.js + Bulma static website about Psychotherapy Kotnik in slovenian language.


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
